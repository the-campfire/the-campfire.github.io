# 기본 명령어

서버에서 자주 쓰는 기본 명령어예요. 명령어는 채팅창에 **`/`** 를 입력하면 목록이 떠요. 🔥

!!! tip "명령어 입력하는 법"
    채팅 입력칸에 `/`를 치면 사용할 수 있는 명령어가 자동으로 나와요. 원하는 명령어를 고르고 빈칸을 채운 뒤 보내면 돼요.

---

## 닉네임

| 명령어 | 설명 |
|---|---|
| `/닉네임변경` | 서버에서 보이는 내 닉네임을 바꿔요. |

- 닉네임은 **7일에 한 번** 바꿀 수 있어요.
- 더 자주 바꾸고 싶다면 상점에서 **닉네임 변경권**을 사서 쓸 수 있어요. → [상점과 꾸미기](shop.md)

---

## 음성방 — `/방`

내가 만든 음성방을 관리하는 명령어예요. **방 안에서** 사용하세요. → [음성방 만들기](voice.md)

| 명령어 | 누가 | 설명 |
|---|---|---|
| `/방 이름 <이름>` | 주인·관리자 | 방 이름을 바꿔요. |
| `/방 인원 <0~99>` | 주인·관리자 | 최대 인원을 정해요. (`0`=무제한) |
| `/방 권한부여 <유저>` | 주인 | 다른 사람에게 방 관리 권한을 줘요. |
| `/방 권한회수 <유저>` | 주인 | 줬던 관리 권한을 거둬요. |
| `/방 권한받기` | 주인·관리자 부재 시 | 남은 사람이 관리 권한을 가져가요. |
| `/방 정보` | 누구나 | 방 정보를 봐요. |

---

화폐·상점 관련 명령어는 [화폐·상점 명령어](commands-economy.md)에서 따로 정리했어요.
