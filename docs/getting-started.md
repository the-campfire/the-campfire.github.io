# 시작하기

타닥불에 들어오신 걸 환영해요! 🔥
입장(규칙 동의·자기소개)은 이미 마치셨죠? 이제부터 서버를 더 재미있게 즐기는 법을 안내해 드릴게요. 생각보다 간단해요.

## 1. 역할 받기

**#역할선택** 채널에서 원하는 역할을 골라요. 골라두면 파티 모집이나 관심사 매칭에 도움이 돼요.

- **게임** — 롤·발로란트·오버워치·배그·롤체·마인크래프트·스팀게임·기타
- **시간대** — 평일 오전/오후/저녁/새벽·주말 (주로 활동하는 때)
- **관심사** — 수다·음악

언제든 다시 바꿀 수 있으니 부담 없이 골라보세요. 자세히는 [역할 안내](roles.md)를 봐 주세요.

## 2. 서버 둘러보기

타닥불에는 즐길 거리가 많아요. 마음 가는 것부터 둘러보세요.

- [채널 안내](channels.md) — 어디서 무슨 얘기를 하는지 한눈에
- [음성방 만들기](voice.md) — 들어가기만 하면 내 방이 자동으로 생겨요
- [파티 모집](party.md) — 같이 게임할 사람을 구해요
- [음악 신청](music.md) — 음성방에서 노래를 함께 들어요

## 3. 코인 모으기

채팅하거나 음성방에 머무르면 **자동으로** 코인과 경험치가 쌓여요. 모은 코인으로 상점에서 꾸미기도 할 수 있어요.

- [코인과 레벨](economy.md) — 어떻게 쌓이고 레벨이 오르는지
- [출석과 미션](attendance-missions.md) — `/출석`·`/미션`으로 받는 추가 보상
- [상점과 꾸미기](shop.md) — 모은 코인으로 색깔·장식 사기
