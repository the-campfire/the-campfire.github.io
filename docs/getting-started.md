# 시작하기

타닥불에 처음 오셨나요? 환영해요! 🔥
입장부터 첫 대화까지, 순서대로 안내해 드릴게요. 생각보다 간단해요.

!!! info "타닥불은 성인 커뮤니티예요"
    타닥불은 **만 20세 이상** 성인만 함께하는 종합 게임·수다 서버예요. 입장 과정에서 출생 연도를 확인해요.

## 1. 규칙에 동의하기

먼저 **#규칙** 채널에서 [서버 규칙](rules.md)을 읽고, **[동의합니다]** 버튼을 눌러주세요.
동의하면 자기소개 채널이 열려요.

## 2. 자기소개 작성하기

**#자기소개** 채널에서 **[자기소개 작성]** 버튼을 누르면, 서버 봇 **불송이**가 단계별로 물어봐요. 모두 4단계예요:

1. 닉네임 + 출생 연도
2. 성별
3. 타닥불을 어떻게 알게 됐는지 (유입 경로)
4. 하고 싶은 활동

!!! note "여기 적는 정보는 비공개예요"
    출생 연도·성별 같은 정보는 **다른 멤버에게 공개되지 않아요.** 운영진이 성인 인증·서버 운영 목적으로만 확인하고, 그 외 용도로 쓰거나 외부에 공유하지 않아요.

자기소개를 끝내면 **@유저** 역할이 자동으로 붙고, 모든 채널이 열려요. 메인채팅에서 환영 인사도 받게 돼요! 🎉

## 3. 역할 받기

**#역할선택** 채널에서 원하는 역할을 골라요. 골라두면 파티 모집이나 관심사 매칭에 도움이 돼요.

- **게임** — 롤·발로란트·오버워치·배그·롤체·마인크래프트·스팀게임·기타
- **시간대** — 평일 오전/오후/저녁/새벽·주말 (주로 활동하는 때)
- **관심사** — 수다·음악

언제든 다시 바꿀 수 있으니 부담 없이 골라보세요.
