# 채널 안내

타닥불 서버는 비슷한 성격의 채널끼리 **카테고리**로 묶여 있어요. 어디서 무엇을 하면 되는지 한눈에 정리해 드릴게요. 🔥

!!! tip "채널 이름 읽는 법"
    채널 이름은 `이모지∙이름` 형태예요. (예: `💬∙메인채팅`) 왼쪽 사이드바에서 이모지를 보고 빠르게 찾을 수 있어요.

---

## 🔥 타닥불

서버 운영과 관련된 안내·신청 채널이 모여 있어요.

| 채널 | 무엇을 하는 곳 |
|---|---|
| `📢∙공지` | 서버 소식·업데이트를 알려요. |
| `📋∙규정` | 규칙 전문(자세한 내용)을 확인할 수 있어요. |
| `🏷️∙역할선택` | 게임·시간대·관심사 역할을 직접 고르는 곳이에요. → [역할 안내](roles.md) |
| `📮∙문의건의함` | 운영진에게 문의하거나 건의할 수 있어요. |
| `🚨∙신고` | 불편한 일이 있을 때 비공개로 신고할 수 있어요. → [신고·문의](support.md) |

## 🌳 커뮤니티

수다 떨고 일상을 나누는, 서버의 중심 공간이에요.

| 채널 | 무엇을 하는 곳 |
|---|---|
| `💬∙메인채팅` | 자유롭게 대화하는 메인 채팅방이에요. |
| `🎬∙쇼츠릴스` | 짧은 영상(쇼츠·릴스)을 공유해요. |
| `🪴∙일상공유` | 일상 이야기를 나누는 포럼이에요. |
| `📸∙짤방` | 재미있는 짤·이미지를 올리는 포럼이에요. |

같이 게임할 사람을 찾고, 게임 이야기를 나누는 곳이에요.

| 채널 | 무엇을 하는 곳 |
|---|---|

| `🎮∙게임추천` | 추천하고 싶은 게임을 소개하는 포럼이에요. |

## 🔊 음성방

목소리로 함께 어울리는 공간이에요.

| 채널 | 무엇을 하는 곳 |
|---|---|
| `🔊∙음성방 생성` | 들어가면 **나만의 음성방이 자동으로 만들어져요.** → [음성방 만들기](voice.md) |
| `🔥∙불멍때리기` | 그냥 켜두고 불멍해도 되는 고정 음성방이에요. 말없이 있어도, 수다를 떨어도 좋아요. |
| `🎮∙같이 게임` | 게임하면서 함께 쓰는 고정 음성방이에요. |

## 👾 봇

| 채널 | 무엇을 하는 곳 |
|---|---|
| `❤️∙불송이` | 서버 관리봇 **불송이**의 명령어를 입력하는 곳이에요. → [기본 명령어](commands-basic.md) |
| `🧮∙출석` | 매일 출석체크를 하는 곳이에요. → [출석과 미션](attendance-missions.md) |
| `📌∙소피가이드` | 카드 수집 봇 **소피(Sofi)** 사용법을 한글로 정리해 둔 곳이에요. |
| `❤️∙소피` | 소피 봇 명령어를 입력하며 카드를 모으는 곳이에요. |

!!! note "소피(Sofi)는 어떤 봇인가요?"
    **소피**는 전 세계에서 쓰이는 카드 수집·가챠 봇이에요. 우리 서버가 만든 봇이 아니라 외부(영어) 봇이라, 명령어가 헷갈리지 않게 `📌∙소피가이드`에 사용법을 한글로 모아 뒀어요.

!!! note "운영 채널은 보이지 않아요"
    로그·관리용 채널은 운영진에게만 보여요. 일반 유저에게 보이는 채널만 위에 정리했어요.
