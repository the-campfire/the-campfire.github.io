# 채널 안내

타닥불 서버는 비슷한 성격의 채널끼리 **카테고리**로 묶여 있어요. 어디서 무엇을 하면 되는지 한눈에 정리해 드릴게요. 🔥

!!! tip "채널 이름 읽는 법"
    채널 이름은 `이모지∙이름` 형태예요. (예: `💬∙메인채팅`) 왼쪽 사이드바에서 이모지를 보고 빠르게 찾을 수 있어요.

---

## 👋 입장

서버에 처음 들어오면 가장 먼저 만나는 곳이에요.

| 채널 | 무엇을 하는 곳 |
|---|---|
| `📋∙입장안내` | 입장 절차를 안내해요. 여기부터 읽어 주세요. |
| `📌∙규칙` | 서버 규칙을 읽고 **[동의합니다]** 를 누르는 곳이에요. |
| `📄∙자기소개` | 동의 후 간단한 자기소개를 작성하면 입장이 완료돼요. |
| `💬∙도와주세요` | 입장하다 막히면 여기에 편하게 불러 주세요. 운영진과 불송이가 바로 도와드려요. |

자세한 입장 절차는 [시작하기](getting-started.md) 페이지를 봐 주세요.

---

## 🔥 타닥불

서버 운영과 관련된 안내·신청 채널이 모여 있어요.

| 채널 | 무엇을 하는 곳 |
|---|---|
| `📢∙공지` | 서버 소식·업데이트를 알려요. |
| `📋∙규정` | 규칙 전문(자세한 내용)을 확인할 수 있어요. |
| `🏷️∙역할선택` | 게임·시간대·관심사 역할을 직접 고르는 곳이에요. → [역할 안내](roles.md) |
| `📮∙문의건의함` | 운영진에게 문의하거나 건의할 수 있어요. |
| `🚨∙신고` | 불편한 일이 있을 때 비공개로 신고할 수 있어요. → [신고·문의](support.md) |

## 🌳 커뮤니티

수다 떨고 일상을 나누는, 서버의 중심 공간이에요.

| 채널 | 무엇을 하는 곳 |
|---|---|
| `💬∙메인채팅` | 자유롭게 대화하는 메인 채팅방이에요. |
| `🎬∙쇼츠릴스` | 짧은 영상(쇼츠·릴스)을 공유해요. |
| `🪴∙일상공유` | 일상 이야기를 나누는 포럼이에요. |
| `📸∙짤방` | 재미있는 짤·이미지를 올리는 포럼이에요. |

## 🎮 게임

| 채널 | 무엇을 하는 곳 |
|---|---|

## 🔊 음성방

| 채널 | 무엇을 하는 곳 |
|---|---|
| `🔊∙음성방 생성` | 들어가면 **나만의 음성방이 자동으로 만들어져요.** → [음성방 만들기](voice.md) |

    로그·관리용 채널은 운영진에게만 보여요. 일반 유저에게 보이는 채널만 위에 정리했어요.
