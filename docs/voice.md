# 음성방 만들기

타닥불에는 **나만의 음성방을 즉석에서 만드는** 기능이 있어요. 친구들과 따로 모이고 싶을 때 채널을 만들어 달라고 부탁할 필요가 없어요. 🔥

---

## 방 만드는 법

1. `🔊 [음성방]` 카테고리의 **`🔊∙음성방 생성`** 채널에 들어가요.
2. 그러면 **나만의 임시 음성방이 자동으로 만들어지고**, 그 방으로 자동으로 옮겨져요.
3. 친구를 초대하거나, 다른 사람이 들어오면 같이 대화하면 돼요.
4. **방에 아무도 없게 되면 방은 자동으로 사라져요.** 따로 지울 필요 없어요.

!!! tip "고정 음성방도 있어요"
    그냥 편하게 들어가 쓰고 싶다면 `☕∙편하게 수다`, `🎮∙같이 게임` 음성방을 이용하세요. 이 방들은 항상 열려 있어요.

---

## 내 방 관리하기 — `/방` 명령어

내가 만든 방의 **주인**은 나예요. 방 안에서 아래 명령어로 방을 꾸밀 수 있어요.

| 명령어 | 누가 쓸 수 있나 | 무엇을 하나 |
|---|---|---|
| `/방 이름 <이름>` | 주인·관리자 | 방 이름을 바꿔요. |
| `/방 인원 <0~99>` | 주인·관리자 | 최대 인원을 정해요. (`0` = 무제한) |
| `/방 권한부여 <유저>` | 주인 | 다른 사람에게 방 관리 권한을 줘요. |
| `/방 권한회수 <유저>` | 주인 | 줬던 관리 권한을 거둬요. |
| `/방 권한받기` | 주인·관리자가 모두 없을 때 | 남은 사람이 스스로 관리 권한을 가져가요. |
| `/방 정보` | 누구나 | 방 주인·관리자·최대 인원·만든 시각을 봐요. |

!!! info "방 주인은 바뀌지 않아요"
    방을 만든 사람이 계속 주인이에요. 다만 주인이 나가고 관리자도 없으면, 남은 사람이 `/방 권한받기`로 관리 권한을 이어받을 수 있어요.

!!! note "이름을 너무 자주 바꾸면 잠깐 막혀요"
    방 이름 변경은 디스코드 자체 제한이 있어서, 짧은 시간에 여러 번 바꾸면 잠깐 기다려 달라는 안내가 떠요. 조금 있다 다시 시도하면 돼요.
