# 서버 규칙

타닥불은 **서로 존중하는 따뜻한 분위기**를 가장 소중하게 여겨요.
모두가 편하게 지내기 위한 약속이에요. 어렵지 않아요. 🙂

## 꼭 지켜주세요

1. **서로 존중하기** — 욕설·비방·과한 훈수는 안 돼요.
2. **불쾌한 언행 금지** — 원치 않는 성적인 언행, 혐오 표현, 분쟁이 되는 주제(정치·젠더 등)는 삼가주세요.
3. **선 넘는 접근 금지** — 원치 않는 DM(개인 메시지), 만남 목적의 접근은 안 돼요.
4. **개인정보 보호** — 다른 사람의 개인정보를 묻거나 퍼뜨리지 마세요.

!!! note "여기 적힌 것 말고도"
    명시되지 않은 일도 분위기를 해친다고 판단되면 운영진이 조치할 수 있어요. 핵심은 '함께 즐겁게'예요.

## 공개 채널에서의 말투 🗣️

공개 채널은 **누구나 편하게 말 걸 수 있는 자리**예요. 그래서 —

- 공개 채널에서는 **기본적으로 다정한 경어체(존댓말)** 로 시작하면 좋아요. 딱딱하게 말고, 이모지도 ㅋㅋ도 농담도 환영이에요 😄
- 친해지면 자연스럽게 말 놓는 건 언제든 좋아요 (음성·파티·DM 등에서요).
- 새로 온 분이 보이면 **먼저 이름 불러 인사 한 번** 건네주세요. 그 한마디가 타닥불에 불씨를 하나 더 올리는 일이에요. 🪵

!!! tip
    반말은 금지가 아니에요. '관계가 익으면 자연스럽게'가 우리 문화예요.

## 규칙을 어기면

상황에 따라 단계적으로 조치해요:

> **구두 경고 → 타임아웃 → 기록 경고 → 서버 차단**

- 사안이 중대하면 단계를 건너뛰거나 즉시 조치할 수 있어요.
- 기록 경고가 **3회 쌓이면 차단**돼요.

## 신고하고 싶을 때

불편한 일이 있으면 **#신고** 채널에서 **[신고 티켓 열기]** 를 눌러주세요.

- 신고자와 운영진만 보이는 **비공개 채널**이 만들어져요.
- 공개 채널에 신고 글을 올리는 건 삼가주세요 (신고자 보호를 위해서예요).
