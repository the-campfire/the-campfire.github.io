# 음악 신청

다 같이 음악을 들으며 어울리고 싶을 때 **`🎵 [음악]`** 카테고리를 이용하세요. 🔥

---

## 듣는 법

1. **`🎧∙DJ빙설`** 음성방에 들어가요.
2. **`🎶∙음악신청`** 채널에서 듣고 싶은 노래를 신청해요.
3. 음악 봇이 신청곡을 순서대로 재생해 줘요.

!!! tip "관심사 역할 @음악"
    음악을 좋아한다면 [역할 안내](roles.md)에서 `@음악` 역할을 켜 두세요. 음악을 즐기는 사람들과 더 잘 어울릴 수 있어요.

!!! note "사용법이 궁금하면"
    음악 봇 사용이 처음이라 막힌다면 `📮∙문의건의함` 채널에 편하게 물어봐 주세요. 운영진이 도와드려요.
