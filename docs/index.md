# 타닥불에 오신 걸 환영합니다 🔥

여기는 **타닥불 서버 위키**예요.
타닥불은 평일 저녁, 함께 게임하고 수다 떠는 **성인(만 20세 이상) 종합 게임·수다 커뮤니티**예요.
서버를 즐기는 데 필요한 모든 것을 여기서 찾을 수 있어요.
