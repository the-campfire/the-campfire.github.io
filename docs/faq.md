# 자주 묻는 질문

궁금한 걸 빠르게 찾아보세요. 각 질문을 누르면 답이 펼쳐져요.

??? question "역할은 어떻게 받나요?"
    **#역할선택** 채널에서 원하는 걸 고르면 돼요.

    - 게임·시간대는 드롭다운(목록)에서 여러 개 선택
    - 관심사(수다·음악)는 버튼으로 켜고 끄기

    언제든 다시 바꿀 수 있어요. 자세히는 [역할 안내](roles.md)를 봐 주세요.

??? question "닉네임은 어떻게 바꾸나요?"
    `/닉네임변경` 명령어로 바꿀 수 있어요. 단, **7일에 1번**만 바꿀 수 있어요.

    - 한글·영문·숫자·공백 사용 가능 (공백 제외 1~12자)
    - 특수문자, 공백만으로 된 닉네임, 운영진·봇 사칭은 안 돼요.
    - 더 자주 바꾸려면 상점의 **닉네임 변경권**을 쓰면 돼요. → [상점과 꾸미기](shop.md)

??? question "코인은 어떻게 모으나요?"
    채팅하거나 음성방에 머무르면 **자동으로** 쌓여요. 따로 명령어를 칠 필요 없어요.

    - **음성 활동이 채팅보다 더 많이** 적립돼요.
    - 도배를 막기 위해 메시지마다 약간의 쿨타임이 있어요.
    - `/출석`·`/미션`으로도 받을 수 있어요. 자세히는 [코인과 레벨](economy.md)·[출석과 미션](attendance-missions.md)을 봐 주세요.

??? question "음성방은 어떻게 만드나요?"
    **🔊 음성방 생성** 채널(허브)에 들어가기만 하면, 내 음성방이 **자동으로 만들어져요.** 방이 비면 자동으로 사라져요.

    - 고정 음성방(편하게 수다 ☕ / 같이 게임 🎮)도 함께 있어요.
    - 방 이름·인원 바꾸기 등은 [음성방 만들기](voice.md)를 봐 주세요.

??? question "자기소개에 쓴 정보는 공개되나요?"
    아니요. 출생 연도·성별 같은 정보는 **다른 멤버에게 공개되지 않아요.** 운영진이 성인 인증·서버 운영 목적으로만 확인해요.

??? question "건의나 문의는 어디에 하나요?"
    - 일반 문의·건의 → **#문의건의함**
    - 입장하다 막히는 게 있으면 → **#도와주세요** (운영진과 불송이가 도와드려요)
    - 신고 → **#신고** (비공개 티켓으로 처리돼요)
