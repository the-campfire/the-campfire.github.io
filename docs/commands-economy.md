# 화폐·상점 명령어

코인·레벨·상점과 관련된 명령어를 모았어요. 🔥

---

## 내 정보 확인

| 명령어 | 설명 |
|---|---|
| `/잔액` | 내 코인 잔액을 봐요. (나만 보기) |
| `/레벨` | 내 레벨과 진행도를 봐요. |
| `/랭킹` | 서버 순위를 봐요. → [랭킹](ranking.md) |

자세한 내용은 [코인과 레벨](economy.md)을 봐 주세요.

---

## 매일 보상

| 명령어 | 설명 |
|---|---|
| `/출석` | 하루 한 번 출석 보상을 받아요. |
| `/미션` | 미션 진행도를 보고 보상을 받아요. |

자세한 내용은 [출석과 미션](attendance-missions.md)을 봐 주세요.

---

## 상점

| 명령어 | 설명 |
|---|---|
| `/상점` | 전체 상품을 둘러봐요. (이름 검색 가능) |
| `/역할상점` | 역할(색깔 등)을 사요. |
| `/닉네임상점` | 닉네임 장식을 사요. |
| `/티켓상점` | 닉네임 변경권 등 티켓을 사요. |

---

## 산 물건 관리

| 명령어 | 설명 |
|---|---|
| `/치장` | 산 역할·닉네임 장식을 착용하거나 해제해요. |
| `/소지품` | 가진 티켓을 확인해요. |

자세한 내용은 [상점과 꾸미기](shop.md)를 봐 주세요.
