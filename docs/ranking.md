# 랭킹

서버에서 누가 활발하게 활동하는지 **`/랭킹`** 으로 볼 수 있어요. 🔥

---

## 🏆 랭킹 보기 — `/랭킹`

랭킹은 **4가지 종류**를 **2가지 기간**으로 볼 수 있어요.

| 종류 | 무엇을 기준으로 |
|---|---|
| **레벨** | 누적 레벨 |
| **채팅** | 채팅 활동량 |
| **음성** | 음성방에 머문 시간(분) |
| **코인** | 가진 코인 |

| 기간 | 설명 |
|---|---|
| **전체** | 지금까지 누적 |
| **월간** | 이번 달 (한국 시간 기준) |

- 각 랭킹은 **상위 10명**을 보여줘요.
- 내가 10등 밖이면 **내 주변 순위(나와 위아래)**도 함께 보여줘서 위치를 알 수 있어요.

!!! info "랭킹에 들어가려면"
    공개 랭킹은 자기소개까지 마친 정식 멤버(`@유저`)를 대상으로 해요. 활동 기록이 있어야 순위에 나타나요.

---

## 내 기록만 보고 싶다면

| 명령어 | 확인 내용 |
|---|---|
| `/잔액` | 내 코인 잔액 |
| `/레벨` | 내 레벨과 진행도 |

코인·레벨이 어떻게 쌓이는지는 [코인과 레벨](economy.md)을 봐 주세요.

!!! tip "채팅 랭킹은 도배해도 안 올라가요"
    채팅 랭킹은 '얼마나 꾸준히 대화했나'를 보는 거예요. 짧은 시간에 도배해도 순위가 오르지 않으니 편하게 활동하세요.
